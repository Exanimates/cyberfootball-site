Сайт "Киберфутбол"
=============================

Сайт посвященный турнирам по киберфутболу. Цель данного сайта создать инфраструктуру для людей увлекающихся киберфутболом

## Установка проекта
```
npm install
```
## Запуск проект
```
npm run start
```
